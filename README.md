# AI Resume Screening System

This project is an AI-powered resume screening system that compares resumes with a job description and ranks them based on relevance using NLP techniques.

##  Features
- Resume and job description matching
- TF-IDF based text vectorization
- Cosine similarity scoring
- Resume ranking system
- Beginner-friendly AI project

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)
- Streamlit (optional UI)
